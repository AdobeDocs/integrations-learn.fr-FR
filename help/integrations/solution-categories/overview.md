# Configurations d’intégration pour Experience Cloud - organisées par solution

Cette section est utile pour toute personne ayant des besoins professionnels, mais ne sait pas exactement quelles applications Adobe utiliser et comment les intégrer. Voici la plupart des configurations d’intégration les plus courantes, organisées par solutions (marketing B2B, supply chain de contenu, informations sur les données et activation, etc.).

Gardez à l’esprit qu’il existe de nombreuses façons de configurer les applications Adobe, selon votre environnement. Cette section a pour but de vous fournir des conseils que vous pouvez déployer en l’état ou modifier en fonction des besoins de votre environnement.
