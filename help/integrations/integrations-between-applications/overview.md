# Configurations d’intégration pour Experience Cloud - organisées par application

Cette section est destinée aux personnes familiarisées avec les applications Adobe et souhaitant créer de la valeur commerciale en intégrant plusieurs applications Adobe. Nous avons organisé de nombreuses configurations d’intégration courantes pour les applications d’entreprise Adobe et nous les avons associées à des cas d’utilisation réels.

Il existe de nombreuses façons de configurer les applications Adobe, selon votre environnement. Cette section fournit des conseils pour vous aider à effectuer un déploiement en l’état ou à les modifier en fonction des besoins de votre environnement.

>[!NOTE]
>
>Utilisez le volet de navigation de gauche pour trouver l’intégration la mieux adaptée à vos besoins.
