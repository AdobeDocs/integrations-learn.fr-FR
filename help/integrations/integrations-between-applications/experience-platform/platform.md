# Intégration d’Experience [!DNL Platform] à d’autres applications Adobe

Adobe Experience [!DNL Platform] est le système ouvert, flexible et puissant du marché permettant de créer et de gérer des solutions complètes qui optimisent l’expérience client. De nombreuses nouvelles applications Adobe ont été créées sur Experience [!DNL Platform], notamment :

* [!DNL Analytics] du Parcours client
* Journey Optimizer
* Real-Time Customer Data [!DNL Platform] (Real-Time CDP)

L’intégration d’applications à _plusieurs applications natives [!DNL Platform]_ partage de nombreuses étapes communes, mais différents cas d’utilisation sont déverrouillés en fonction des applications que vous intégrez. Le contenu de l’intégration est donc organisé par application. Consultez les ressources ci-dessous.
